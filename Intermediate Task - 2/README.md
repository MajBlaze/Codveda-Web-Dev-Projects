# 📝 To-Do App (Internship Project)

Hey! I built this To-Do App as part of my internship work.
I wanted to create something simple, clean, and actually useful while improving my frontend skills.

## 🚀 Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Progress bar with completed/total count
- Data saved in localStorage
- Confetti when all tasks are completed
- Responsive design for mobile

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome
- canvas-confetti

## 📂 Project Structure

```text
To-Do list using Html , Css & JS/
├── index.html
├── style.css
├── script.js
└── images/
    ├── background.jpg
    └── image.png
```

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No build tools required.

- Handling user events (submit, edit, delete, checkbox)

- Building responsive UI with media queries
- Making UI interactions smoother and cleaner
