# Intermediate Task 3 - React Calculator

This is my Level 2 Task 3 project.

I made a calculator app using React to practice components, props, and state.

## Features

- Add, subtract, multiply, divide
- AC (clear all)
- DEL (delete one digit)
- Decimal input
- Small recent history section
- Responsive layout for mobile and desktop

## React concepts used

- Components:
`CalculatorDisplay`, `CalculatorButtons`
- Props:
Passing button click handler and display values
- State:
Screen value, first number, selected operator, waiting flag, history list

## Tech used

- React (Create React App)
- JavaScript
- HTML
- CSS

## Run project

1. Open terminal in this folder: `Intermediate Task - 3`
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

## Build project

Run `npm run build`

## My learning

I understood better how to split one app into small components and connect them with props.
I also learned state handling in a practical way while writing calculator logic.
