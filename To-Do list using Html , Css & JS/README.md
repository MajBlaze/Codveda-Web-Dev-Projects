# 📝 To-Do App (Internship Project)

Hi! I am a fresher developer and this is my **To-Do List Web App** project built during my internship task.  
I made this project to practice real frontend development with clean UI, JavaScript logic, and responsive design.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed
- 📊 Progress bar with completed/total task count
- 💾 Saves tasks in `localStorage` (data stays after refresh)
- 🎉 Confetti effect when all tasks are completed
- 📱 Responsive layout for mobile screens

## 🛠️ Tech Stack

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla JS)`
- `Font Awesome` icons
- `canvas-confetti` library

## ▶️ How to Run

1. Download or clone this repository.
2. Open this project folder.
3. Run `index.html` in your browser.

That’s it, no build tools needed. 🙂
